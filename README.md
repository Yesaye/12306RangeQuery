# 12306RangeQuery
12306买长乘短 买短乘长查询（为了铁路出行安全，建议优先考虑买长乘短）

# 说明

暂时一个小demo，够我用了， 后续有时间再完善

# 使用方法

1. java基础：略
2. 引用包：略
3. 修改参数
   1. cookie：网页登录12306，随便搜索一个，将cookie复制过来
   2. ua: 可以改可不改，获取方法同上
   3. 修改main方法中的from，to，date，稍等片刻会在控制台输出结果，这里的站点为精确查询，不包含同城车站
4. 按照结果找有票的去搜索买票即可
